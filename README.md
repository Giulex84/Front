# PactPI

PactPI is a web application designed for the Pi Network ecosystem that allows users to create and manage mutual digital commitments.

The app focuses on transparency, responsibility, and trust between participants. PactPI is not a financial service and does not facilitate payments, transfers, or legal contracts.

## Key principles

- Commitments are voluntary and based on mutual consent
- Trust signals are informational and derived from observable activity
- No financial guarantees or enforcement mechanisms are provided
- The app is designed to discourage misuse and impulsive agreements

## Status

PactPI is currently in early development and review phase within the Pi Network ecosystem.

## Technology

- Frontend: React + TypeScript + Vite
- Hosting: Vercel
- Backend services (when enabled): Supabase

## Disclaimer

PactPI does not provide financial services, investment opportunities, or legally binding agreements. All interactions are voluntary and informational.
